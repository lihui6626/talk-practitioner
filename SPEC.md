# 接话与逻辑表达练习器 - 产品设计文档

## 1. 产品概述

**产品定位**: 一款专注于职场沟通与表达的能力提升工具，通过真实场景模拟和梯度选项练习，帮助用户掌握自然接话技巧和结构化表达能力。

**核心价值**: 
- 碎片化时间练习，随时随地提升沟通能力
- 场景化学习，避免空洞的理论
- 深度分析反馈，不止于判断对错

## 2. 设计风格

**视觉风格**: 极简主义
- 主色调: 深灰 #2C3E50 + 亮橙 #E67E22（用于强调和得分）
- 背景: 纯白 #FFFFFF 或浅灰 #F8F9FA
- 字体: 系统默认，清晰易读
- 间距: 宽松留白，减少视觉压力
- 图标: 线性图标，极简风格

## 3. 功能模块

### 模块一: 日常接话练习

**场景类型**:
1. 电梯偶遇 - 30秒内的简短对话
2. 茶水间闲聊 - 轻松的社交场合
3. 食堂搭话 - 边吃边聊的自然场景
4. 走廊相遇 - 偶遇领导的应对

**练习特点**:
- 场景真实，还原度高
- 选项体现不同的接话深度
- 重点: 共情力、话题延展、自然过渡

### 模块二: 职场逻辑表达

**场景类型**:
1. 向上汇报 - 数据驱动的结构化汇报
2. 需求评审 - 清晰表达方案价值
3. 面试追问 - 展现逻辑思维深度
4. 跨部门协作 - 有理有据的沟通

**练习特点**:
- 高压场景模拟
- 选项体现结构化思维差异
- 重点: 逻辑清晰、数据支撑、观点明确

## 4. 题目设计

### 选项梯度设计（4级）

**日常接话 - 评分标准**:
- A (2分): 完全跑偏，让对话陷入尴尬
- B (5分): 勉强接住但话题延展差
- C (8分): 自然接话，能延展但深度不够
- D (10分): 高情商接话，话题延展自然有深度

**职场逻辑 - 评分标准**:
- A (2分): 逻辑混乱，抓不住重点
- B (5分): 有一定逻辑但结构松散
- C (8分): 结构清晰但说服力一般
- D (10分): 逻辑严谨，数据支撑强，有说服力

### 分析维度

**日常接话分析**:
1. 情绪感知: 是否准确捕捉对方情绪
2. 话题承接: 是否自然过渡
3. 话题延展: 是否创造继续聊下去的空间
4. 情商体现: 是否得体、让人舒适

**职场逻辑分析**:
1. 结论先行: 是否先说结论
2. 逻辑结构: 是否使用金字塔原理
3. 数据支撑: 是否有数据/事实依据
4. 预判能力: 是否预判并回应质疑

## 6. 交互流程

1. **首页**: 两个模块入口，简洁明了
2. **场景选择**: 每个模块下选择具体场景
3. **练习页面**: 显示场景和4个选项
4. **选择确认**: 点击选项后确认选择
5. **结果页面**: 显示选择得分 + 全选项分析
6. **继续/返回**: 可选择下一题或返回

## 7. 技术实现

**框架**: 微信小程序原生开发 (TDesign 组件库)
**状态管理**: 简单状态管理，无需复杂方案
**数据存储**: 本地存储练习记录和历史
**无需后端**: 纯前端应用
