# 接话与逻辑表达练习器

一款专注于职场沟通与表达的能力提升工具，通过真实场景模拟和梯度选项练习，帮助用户掌握自然接话技巧和结构化表达能力。

## 产品特点

### 🎯 两个核心模块

**日常接话练习**
- 电梯偶遇
- 茶水间闲聊
- 食堂搭话
- 走廊相遇

**职场逻辑表达**
- 向上汇报
- 需求评审
- 面试追问
- 跨部门协作

### 📊 科学的评分体系

每题4个选项，从最差到最优有明显梯度：
- **A (2分)**: 典型错误示范
- **B (5分)**: 基本合格但不够好
- **C (8分)**: 良好，有亮点
- **D (10分)**: 优秀，教科书级别

### 💡 深度分析反馈

**日常接话分析维度**：
- 情绪感知
- 话题承接
- 话题延展
- 情商体现

**职场逻辑分析维度**：
- 结构清晰（结论先行）
- 数据支撑
- 逻辑严谨
- 预判能力

## 设计理念

### 极简风格
- 清晰的视觉层次
- 舒适的配色方案
- 宽松的留白设计

### 真实场景
- 所有场景均来自真实职场
- 选项设置贴合实际对话
- 分析有干货，不敷衍

### 刻意练习
- 碎片化时间即可练习
- 每题都是独立的知识点
- 循序渐进提升能力

## 使用建议

1. **每天练习1-2题**：利用碎片时间，保持手感
2. **先思考再选择**：不要急于看答案，先自己思考
3. **分析比分数重要**：理解底层逻辑才是关键
4. **应用到实际**：练习后找机会在实际场景中运用
